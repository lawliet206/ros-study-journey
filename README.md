# ros-study-journey
记录本科四年关于电子相关的学习
本来打算在大一上半年刚开始学 Arduino 时就着手丰富自己的 GitHub 主页，可惜一直没能注册成功。最近在准备无人机比赛的视觉识别部分，用 GitHub 的频率不可避免地在增加，所以今天下定决心，想办法注册了一个账号。

目前的近期小目标是先想办法把无人机视觉识别搞定，但眼下训练出来的模型精度很不堪入目。不过，我觉得前两天那个卖源码的人说得挺对——训练集太模糊，常规模型基本都不太好用，应该把重心转移到视觉处理上，只是视觉处理确实有难度。稍微远期一点的目标，是最终考核要用到的那个 ROS 小车，我想尽量把它做好一点，直接当成自己的毕设来做。
